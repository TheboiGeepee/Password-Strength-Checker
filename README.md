# 🔐 Real-World Password Checker

An Interactive Python-based password strength checker designed to analyze password security using:

- Password complexity analysis
- Common password detection
- Pattern detection
- Wordlist comparison
- Estimated crack time
- Optional HaveIBeenPwned API support

Built with a hacker-style terminal interface.
